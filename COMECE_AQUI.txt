❓ PROBLEMAS?

  ❌ "Arquivo Python não encontrado"
     → Instale Python 3.9+ de https://python.org

  ❌ "ModuleNotFoundError: No module named..."
     → Execute: pip install -r requirements.txt

  ❌ "msedgedriver not found"
     → Execute: python prepare_drivers.py

  ❌ Compilação muito lenta (>10 min)
     → Normal! Primeira vez demora mais. Seja paciente.

  ❌ Outras dúvidas
     → Consulte GUIA_COMPILACAO.md
